Help & Support

Get help with using TapIn

  1. Create an account or log in
  2. If you're a student, join a class using the PIN provided by your teacher
  3. If you're a teacher, create classes and set up geo-fences
  4. Use the check-in feature when attending classes

TapIn uses GPS to verify your location for attendance. Make sure:

  • Location services are enabled on your device
  • You grant location permission when prompted
  • You're within the geo-fence area set by your teacher
  • GPS accuracy is good (avoid indoor locations if possible)

Can't check in?
  • Ensure location services are enabled
  • Check if you're within the class geo-fence
  • Try refreshing the page and trying again
Location permission issues?
  • Check browser settings for location permissions
  • Try using a different browser
  • Ensure you're using HTTPS (required for location)
